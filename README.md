# quiz-animated
Trivia quiz practice - animated cartoons

My second pub trivia quiz app

This was built using Claude, and was intended as a downloadable, offline app for phones, when you're just hanging around waiting. You can go ahead and do that - I wrote a blog post at https://www.geekmum.com.au/building-app-claude/ on how I added it to my Google phone. It will be slightly different for iPhones.
As there is Javascript embedded in the HTML file, you may need to tweak the setup to allow JS to run.

Or you can access it and play at  https://ozgeekmum.github.io/quiz-animated/

Enjoy!
